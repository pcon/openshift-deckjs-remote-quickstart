How to use

  1. Follow the quickstart documentation
  2. Launch the presentation with the parameter ?master
  3. Enter password of "master"
  4. Wait for your guests to arrive and choose "Join" on the presentation page
  5. Profit!

How to change the default password

  1. Get the md5sum hash for your password echo -n "mynewpassword" | md5sum
  2. Take that hash and update extensions/remote/deckjs-remote.js with the new key on line 13
